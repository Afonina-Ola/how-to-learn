# Научиться учиться

## О проекте:

Одностраничный сайт о современных подходах к обучению для людей разного возраста.

## Функциональность:

- анимации,
- трансформации,
- iframe,
- ссылки на страницы.

## Технологии:

- БЭМ,
- флексбокс-верстка.

## Инструкция по локальному запуску:

- скачать или склонировать репозитрий,
- открыть файл index.html.
